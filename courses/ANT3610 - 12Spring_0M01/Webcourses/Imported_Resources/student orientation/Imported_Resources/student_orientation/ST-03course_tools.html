Course Navigation

Course Tools

Any tools that your instructor has added to the course appear under Course Tools and My Tools. The Course Tools and My Tools menus always appear in the left-hand frame of any page you navigate to in the course.

Course Tools

When you access a tool from Course Tools or My Tools, you are taken to the main screen for that tool. For example, if you access the Discussions tool from Course Tools, you are taken to the main screen for the Discussions, where you can see all of the course discussion topics, including any discussions that have been added to the Course Content Home or other learning modules in the course.

If new content has been added to a tool, an asterisk appears next to the tool name.

Breadcrumbs

As you navigate from the Course Content Home to the individual tools and other content that has been added to the course, your progress through each page is tracked through Breadcrumbs. The Breadcrumbs always appear at the top of every page you visit in your course.

The Breadcrumbs

The Breadcrumbs allow you to retrace your steps through the course by providing a link to previously visited pages; clicking the link returns you to that page.
