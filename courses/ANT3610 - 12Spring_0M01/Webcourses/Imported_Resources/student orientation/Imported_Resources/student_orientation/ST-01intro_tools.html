Getting Started with Your Online Course

Welcome to Online Courses

Every online course is unique. For example, your instructor may be using Webcourses@UCF to supplement a course taught in a regular classroom environment and may only add a couple of key tools to the course. Alternatively, your course may be taught entirely online and a variety of tools and content may be added to your online course.

UCF provides the Learning Online Web site to help you in your online experiences. There are many resources and helpful hints and tips available at this Web site. If you have questions about online learning, check there first. The address is http://learn.ucf.edu.

The following module will assist you in learning about the latest upgrade of UCF's course management system.

The Webcourses@UCF Tool Set

Online courses have a variety of tools that are used to create a learning environment that closely mirrors a regular classroom environment. Essentially, there are three types of Webcourses@UCF tools: Course Content Tools, Communication Tools, and Evaluation Tools.

Course Content Tools

Course Content tools are used to deliver course material, such as the course syllabus, online lectures, and relevant web links, to you. Some of the Course Content tools include:

  • Course Content Home [Webcourses@UCF Homepage in previous version]
  • Learning Modules [Modules, Activities, Materials, Lessons, etc.]

Communication Tools

Communication tools allow you to communicate with your classmates and instructor. Some communication tools are used to facilitate class discussions while others are used to allow for more private communication. Some of the Communication tools include:

  • Discussions
  • Chat
  • Mail [Course Mail in previous version]

Evaluation Tools

Evaluation tools are used to help you gain a better understanding or test your knowledge of the course material. The evaluation tools include:

  • Assignments [also known as Drop Box, etc.]
  • Assessments [Quizzes, Surveys, etc.]

If you encounter questions while using Webcourses@UCF, you can access the online help by clicking the Help link. The link appears in the top right-hand frame of every page. Just in time help is provided for the component you are currently viewing.

Because each course is unique, the tools you use and how you navigate your online course will depend on how it's been set up. However, there are some features that are commonly used to organize course content and to allow you to access tools. Let's get started by taking a look at these common features.
