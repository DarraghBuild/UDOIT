Navigating Chat

When you access Chat from Course Tools, you are taken to the main screen for the Chat tool, which contains all of the chat rooms that are currently available to you.

The Chat Screen

The Chat screen provides you with information about each room, including:

  • the room title.
  • a description of the room.
  • the number of participants currently in the room.

To enter a chat room, click its title or the Go to Chat Room icon in its ActionLinks menu. The chat room opens in a pop-up window.

 

Chat Room

Troubleshooting

You may encounter problems accessing chat/whiteboard rooms if:

  • you don't have the correct version of Java installed.
  • you have software that blocks pop-up messages.

For more information on how to resolve these issues, please visit Webcourses@UCF Browser Tuneup page.
