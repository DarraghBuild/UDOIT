Navigating Assessments

When you access the Assessments tool from Course Tools in the left panel, you are taken to the main screen for the Assessments tool, which lists all of the assessments that are currently available to you.

The Assessments Screen

Below the assessment title, you'll see the start time for the assessment, and, possibly, a brief description of the quiz. If you haven't yet viewed the assessment, the title appears in bold. When you click an assessment title, an information page for the assessment opens.

Assessment Information

The assessment information screen lists the assessment's start and end times, how long you have to complete the assessment, how questions will be delivered to you, the number of attempts allowed, and, if applicable, the maximum grade. If you have more questions about the assessment or technical concerns, click Instructions and Troubleshooting. To begin the assessment, select the Begin Assessment button.
