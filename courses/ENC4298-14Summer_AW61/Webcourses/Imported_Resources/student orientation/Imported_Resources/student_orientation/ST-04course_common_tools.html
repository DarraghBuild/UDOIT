Common Tools

Some tools are used throughout your online course. For example, you may be required to upload files or create Web pages within a variety of Webcourses@UCF tools.

Adding Attachments

While using various online tools, you may need to upload files from your local computer to your course. For example, to submit an assignment, you may need to upload a Microsoft Word file that contains your submission, or you may want to attach a file to a Mail message.

If uploading a file is an option within the tool you are using, you will see an Add Attachments button. In this example, when creating Mail messages, the Add Attachments button appears at the bottom of the Create Message screen.

The Create Message Screen

When you click Add Attachments, the Content Browser appears in a pop-up window.

The Content Browser

To upload files to your course, from the Content Browser, click the My Computer icon and browse your local computer for the files you need to upload. You can upload multiple files at once or individually.

The Content Browser

Once uploaded, the files appear in your My Files folder. To attach the file, select it and click OK.

The HTML Creator  [previously HTML editor]

The HTML Creator allows you to format files in HTML. For example, you could use the HTML Creator to create the pages of a Web site assignment. If the HTML Creator is available within a tool, an On/Off toggle appears above the provided text box. If you select "On," the HTML Creator formatting tools appear in a tool box above the text box.

HTML Creator
